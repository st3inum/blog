Schema.org Structured Data Validation

Validation Tools

Use these tools to validate the schema.org structured data on your blog:

  • Google Rich Results Test
  • Schema.org Validator
  • Google Search Console

Pages to Validate

1. Homepage

URL: https://steinum.dev

Expected Schema Type: WebSite

  • Name property is present ("steinum's blog")
  • URL property is valid
  • Author/publisher properties reference the Person entity
  • SearchAction is properly configured

2. About Page

URL: https://steinum.dev/about

Expected Schema Type: Person

  • Name property is present ("Fahim Tajwar Saikat")
  • AlternateName property is present ("steinum")
  • URL property is valid
  • Social profiles are correctly formatted
  • Education, work experience, and skills are included

3. Blog Post Page

URL: https://steinum.dev/posts/Intro-to-modular-arithmetic

Expected Schema Type: BlogPosting

  • Headline property is present
  • Author property references the Person entity
  • DatePublished property is in ISO format
  • Image property is valid (if present)
  • Publisher property references the Person entity
  • MainEntityOfPage property is correctly configured

How to Validate

  1. Copy the URL of the page you want to validate
  2. Open the Google Rich Results Test tool
  3. Paste the URL and click "TEST URL"
  4. Review the detected structured data types
  5. Check for any errors or warnings
  6. Verify that all required properties are present

Sample JSON-LD Implementations

Person Schema

{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "Fahim Tajwar Saikat",
  "alternateName": "steinum",
  "url": "https://steinum.dev/about",
  "sameAs": [
    "https://github.com/st3inum",
    "https://www.linkedin.com/in/fahim-tajwar/"
  ],
  "jobTitle": "Software Engineer",
  "worksFor": {
    "@type": "Organization",
    "name": "Company Name"
  }
}

BlogPosting Schema

{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "Post Title",
  "description": "Post description",
  "image": "https://steinum.dev/image.jpg",
  "datePublished": "2023-01-01T00:00:00Z",
  "dateModified": "2023-01-01T00:00:00Z",
  "author": {
    "@type": "Person",
    "name": "Fahim Tajwar Saikat",
    "url": "https://steinum.dev/about"
  },
  "publisher": {
    "@type": "Person",
    "name": "Fahim Tajwar Saikat",
    "url": "https://steinum.dev"
  },
  "mainEntityOfPage": {
    "@type": "WebPage",
    "@id": "https://steinum.dev/posts/post-slug"
  }
}
